# Homework-Submission-Database
A simple Python-based application to manage and track homework submissions using a MySQL database.

# 🛠 Features
 <br>--> __Automated Table Creation:__ Checks for and creates the required database schema automatically.</br>
 --> __Add Submissions:__ Easily log student names, homework titles, submission dates, and content.
 <br>--> __View Records:__ Retrieve and display all stored homework submissions from the database.</br>
