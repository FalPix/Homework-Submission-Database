# Homework-Submission-Database
A simple Python-based application to manage and track homework submissions using a MySQL database.

# 🛠 Features
 --> Automated Table Creation: Checks for and creates the required database schema automatically.
 --> Add Submissions: Easily log student names, homework titles, submission dates, and content.
 --> View Records: Retrieve and display all stored homework submissions from the database.

 Prerequisites
